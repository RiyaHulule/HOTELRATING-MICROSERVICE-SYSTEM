1. Microservices Application – Spring Boot, MySQL & MongoDB

   A microservice-based application built using Spring Boot, connected to MySQL and MongoDB. It includes services for user details, hotel information, and ratings, all integrated through an API Gateway.

2. Key Features

- Microservices architecture (User, Hotel, Rating services)

- API Gateway for routing

- Fetch user-specific hotel ratings and details

- MySQL + MongoDB database integration

- Distributed data handling between services

3. Tech Used

   Java 8, Spring Boot, Microservices, API Gateway, MySQL, MongoDB, REST API, Spring Security

4. How to Run

- Configure MySQL & MongoDB details in each service’s application.properties

- Start individual microservices

- Start the API Gateway

- Access services through the Gateway URL

5. Outcome

   This project demonstrates building scalable microservices, implementing inter-service communication, distributed databases, and secure REST APIs using Spring Boot.
